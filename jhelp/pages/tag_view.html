Tagübersicht
 
 
 

Die Tag-Ansicht gibt die Möglichkeit, importierte Gruppen, die in DS Workbench als Tags bezeichnet werden, zu konfigurieren. Hier kann man die Tags umbenennen oder ihr auf der Karte eingezeichnetes Symbol verändern. Dafür ändert man die Werte direkt in der Tabelle, woraufhin sie direkt auf die Karte übertragen werden. Auf der rechten Seite findet man mehrere Button die dazu dienen, einen neuen Tag zu erstellen, Tags zu verknüpfen (worauf später eingegangen wird), einen bestehenden Tag zu löschen, die Farbmarkierung auf der Karte zu entfernen oder gewählte Tags auf verschiedene Arten in die Zwischenablage zu exportieren. Das Entfernen des zu zeichnenden Symbols auf der Karte kann man direkt über die Tabellenzelle bewerkstelligen.
Zudem kann man in der Tag-Tabelle angeben, ob der Tag auf der Karte eingezeichnet werden soll oder nicht. Letztendlich habt ihr die Möglichkeit über den letzten Button im oberen Bereich die markierten Tags und ihre zugeordneten Dörfer als BB-Code in die Zwischenablage zu kopieren, um eventuell bestimmte markierte Dörfer euren Stammeskollegen zugänglich zu machen.

Im unteren Bereich sieht man eine Liste der Dörfer, die dem aktuell markierten Tag/den aktuell markierten Tags zugeordnet sind. Sind mehrere Tags markiert, werden alle Dörfer aus allen Tag-Gruppen angezeigt. Über den oberen Button neben der Liste kann man für das markierte Dorf alle markierten Tags entfernen. Der untere Button dient dazu, das gewählte Dorf auf der Hauptkarte zu zentrieren.

 

Tags verknüpfen

 
 
Durch die Möglichkeit Tags zu verknüpfen erhaltet ihr ein mächtiges Werkzeug um eure Gruppen im Spiel noch besser zu nutzen. Auf den ersten Blick schaut dieses Werkzeug recht komplex aus, allerdings werdet ihr schnell merken dass es das gar nicht ist.
