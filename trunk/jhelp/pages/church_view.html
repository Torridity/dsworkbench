Kirchenübersicht
 
 
Die Kirchenübersicht ist verfügbar, wenn man einen Servern ausgewählt hat, der das Kirchensystem unterstützt. Hier können den Dörfern in denen über das entsprechende Werkzeug eine Kirche platziert wurde, entfernt oder auf der Karte zentriert werden. Die entsprechenden Kirchenradien werden aus der Hauptkarte eingezeichnet, sofern man im Hauptfenster im Bereicht Kartenanzeige Kirchenradien einzeichnen aktiviert hat. Kirchen können in DS Workbench in beliebigen Dörfern platziert werden. Die Farbe des Kirchenradius entspricht dabei der Farbe der Markierung des Spielers in dessen Dorf die Kirche gesetzt wurde. Farbmarkierungen können mit der entsprechenden Ansicht oder mit dem entsprechenden Werkzeug gesetzt bzw. geändert werden.
Kommt es auf der Karte zu Überscheidungen von Kirchenradien die demselben Spieler zugeordnet sind, so werden sie zu einem zusammenhängenden Gebiet kombiniert, wodurch man schnell die Abdeckung einer Region erkennen kann und auch prüfen kann, ob eine Kirche mit einer bestimmten Größe ein anderes Dorf mit einschließt.
